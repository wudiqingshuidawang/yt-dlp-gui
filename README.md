# SakuraFetch ✿

一个可爱的动漫风格视频下载工具，基于 yt-dlp 构建。

## ✨ 特性

- 🌸 动漫风格界面，樱花主题
- 🎬 支持 YouTube、Bilibili 等主流视频平台
- 📺 视频/音频/字幕多种下载模式
- 🎵 音频格式转换 (MP3, AAC, FLAC, WAV)
- 📝 字幕下载与嵌入
- 📋 下载历史记录
- ⚡ 多线程并发下载

## 📦 安装

### macOS
下载 `.dmg` 文件，拖入 Applications 文件夹即可。

### Windows
下载 `.msi` 安装包，双击安装。

### Linux
下载 `.AppImage` 或 `.deb` 文件。

## 🛠️ 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建生产版本
npm run tauri build
```

## 📋 依赖

- [yt-dlp](https://github.com/yt-dlp/yt-dlp) - 视频下载引擎

动漫风格界面，包含樱花飘落、星星闪烁等装饰效果。
